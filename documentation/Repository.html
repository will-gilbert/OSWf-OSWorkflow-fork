OSWf Repository

The OSWf binaries are avaliable as either references in a Maven2 pom.xml file or Gradle build.gradle or as a direct download of the JAR file.

Access the Repository from a Maven POM

These libraries can be accessed via the POM by reference the following Maven repository by adding a repositories tag to your POM. 

    <repository>
      <id>Informagen</id>
      <url>http://informagen.org/maven</url>
        .
        .
        .
    </repository>

Access the Repository from a Gradle Build File

These libraries can be accessed via Gradel by referencing the following Maven repository by adding to your repositories configuration in your gradle.build. 

        repositories {
           // Informagen repository
           mavenRepo urls: ["http://informagen.org/maven/"]
            .
            .
            .
          mavenLocal();
          mavenCentral();
        }

OSWf - Core

Use a Maven dependency

    <dependency>
      <groupId>net.sf.oswf</groupId>
      <artifactId>oswf-core</artifactId>
      <version>3.0.0-SNAPSHOT</version>
    </dependency>

Or Gradle dependency

 "net.sf.oswf:oswf-core:3.0.0-SNAPSHOT"

OSWf - Store

Use a Maven dependency

    <dependency>
      <groupId>net.sf.oswf</groupId>
      <artifactId>oswf-store</artifactId>
      <version>3.0.0-SNAPSHOT</version>
    </dependency>

Or Gradle dependency

 "net.sf.oswf:oswf-store:3.0.0-SNAPSHOT"
